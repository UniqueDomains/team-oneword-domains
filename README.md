## 🧱 Field summary

- `domain` — Fully qualified domain name.
- `status` — Current acquisition state for the domain in the public extract.
- `purchase_price` — Visible purchase price when available.
- `renewal_price` — Visible renewal price when available.
- `attractiveness` — Composite naming score used as a decision-support signal.
- `demand` — Relative buyer-pressure score when available.
- `length` — Character count without the TLD.
- `registrar` — Registrar name when known.
- `created_at` — Creation timestamp when known.
- `expires_at` — Expiry timestamp when known.

## ⚠️ Methodology and caveats

These domains are all one-word names on the .team extension, which gives the set a clear collaborative and group-oriented positioning. The range runs from broad, simple words such as cat.team, goal.team, just.team, and make.team to more opinionated choices like yuck.team, careful.team, and steak.team. When comparing these domains, focus on whether the word benefits from the .team ending rather than merely tolerating it. Stronger picks usually read naturally as a unit, are easy to say and spell, and stay memorable without needing explanation. With a median ask of 11.43, price discipline still matters, especially when choosing between novelty and long-term brand clarity.

- Prefer words that read naturally with .team
- Check spelling clarity and spoken recall
- Use tone carefully: playful words can narrow fit
- Compare ask price against long-term brand strength
